## ℹ️ 公開ポリシー

- このリポジトリは **公開（public）** です。受講生の個人情報・提出物・有料講座の教材本文・請求書などは **一切含みません**。
- 「Dify 課題レビュー資料」は、提出課題を **氏名・ハンドル名を伏せて匿名化** したうえで内容のみを抜粋・解説した学習用資料です。
