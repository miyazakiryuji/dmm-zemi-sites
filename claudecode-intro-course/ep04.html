EP04
料金プランの選び方

無料・個人向け・法人向けの違いを整理して、自分に合ったプランの判断材料をそろえます。

Claude Code を本格的に使い始める前に、まず気になるのが「お金はどれくらいかかるのか」「どのプランを選べばいいのか」という点です。この回では、Anthropic（アンソロピック）の公式料金ページを手がかりに、無料プラン・個人向け有料プラン・法人向けプランの違いをやさしく整理します。講座でどのプランを前提に進めるかも確認し、自分にとって課金が必要かどうかを落ち着いて判断できる状態を目指しましょう。

Learn

この回の学習ノート

1プランは大きく3つに分かれる

Claude の料金プランは、ざっくり「無料で試せるプラン」「個人向けの有料プラン」「会社・組織向けのプラン」の3つに分けて考えると整理しやすくなります。公式の料金ページを開くと、これらが並んで表示され、それぞれに含まれる機能の一覧も確認できます。

  • 無料プラン：費用をかけずに Claude のチャット機能を試せる入り口。まずは触ってみたい人向け。
  • 個人向け有料プラン：Pro と Max の2種類。日々の作業に使う人から、たくさん使いたい人まで段階がある。
  • 法人向けプラン：Team と Enterprise。複数人での利用や、より大きな組織での導入を想定したプラン。

最初から完璧に覚える必要はありません。「個人で学習するなら無料か個人向け有料、会社で使うなら法人向け」という大枠だけ押さえておけば十分です。

2無料プランでは Claude Code は使えない

ここが少しややこしいポイントです。費用をかけないプランでも Claude とのチャットそのものは体験できますが、この講座で手を動かしながら触れていく Claude Code のほうは、無料の範囲では動かせません。別の公式ドキュメントを見ると、Claude Code を動かすには Pro・Max・Team・Enterprise のいずれかの継続契約が前提になる、と書かれています。

そういう事情があるため、この講座では手を動かす練習をつまずかずに進められるよう、原則として Pro プランをベースに解説していきます。

「Claude（チャット）は無料で試せる」けれど「Claude Code は有料プランが必要」という区別が、この回でいちばん大事なポイントです。同じ Claude でも、機能によって必要なプランが違う点を覚えておきましょう。

3課金は必須ではない — 自分のタイミングで判断する

「講座で Pro プランを使う」と聞くと、すぐに有料契約しなければと感じるかもしれませんが、ここで必ず課金が要るわけではありません。お金をかけずに体験できる範囲だけ触ってみる、あるいは操作はせず動画を見る側に回って全体像をつかむ、といった進め方でもまったく差し支えありません。

ひととおり内容を眺めたうえで「自分の手で何か作ってみたい」「Claude Code を動かしてみたい」と感じたら、その段階で契約するかどうかを決めれば十分です。判断のものさしは、目的・どれくらいの頻度で使うか・予算、そして個人で使うのか会社で使うのか。これらに照らして自分に合うものを選びましょう。

迷ったら「視聴で全体像をつかむ → やってみたくなったら契約」の順がおすすめです。先に課金して放置するより、必要だと感じてから契約したほうが無駄がありません。

4料金はドル建て — 日本円はあくまで目安

料金ページの金額がドル建てで出ていることがあります。その場合、日本円にするといくらになるかは為替次第で動くので、画面に出てくる円の目安はあくまで参考程度に見ておきましょう。

個人向けでは Pro（日々の作業向け）と、より多くの使用量を求める人向けの上位プランである Max があります。年払いを選ぶと割引が効くなど、支払い方によっても総額は変わってきます。具体的な金額は時期や為替で動くので、申し込む前には公式ページの表示と決済画面に出る金額を、必ず自分の目で確かめてください。

料金・機能・利用条件・画面表示は日々アップデートされます。動画と実際の画面が違って見えることもあるので、契約前は必ず公式ページの最新情報を確認しましょう。

5会社で使うなら料金以外の確認も必要

会社や組織で導入するなら、Team・Enterprise といった法人向けプランが選択肢になります。料金は「シート」、つまり1人が使うための枠を単位に決まり、たとえば5枠を契約すれば5人が使える、というイメージです。要するに使う人数が増えれば、その分だけ料金も変わる仕組みです。Enterprise はより規模の大きい会社や組織を想定したプランにあたります。

法人で使うときに特に気をつけたいのは、料金よりもむしろデータの扱い、いわゆるプライバシーポリシーです。プライバシーポリシーとは、個人情報やデータをどう扱うかという方針をまとめた文書のこと。入力した内容がどう保管されるのか、AI の学習に回されるのかどうかは、業務で使う以上きちんと押さえておきたいところです。自分の勉強のために使う場合と、会社の仕事で使う場合とでは、見ておくべき観点がそもそも違ってきます。

会社・組織で導入を検討するなら、料金表だけで判断せず、公式サイトのプライバシーポリシーや関連ページを必ず確認しましょう。データの扱いは、料金以上に重要な判断材料になることがあります。

Key Points

この回のポイント

  • プランは「無料」「個人向け有料（Pro／Max）」「法人向け（Team／Enterprise）」の3つに大別される
  • 無料プランでチャットは試せるが Claude Code は使えず、利用には Pro 以上の有料サブスクリプションが必要
  • 講座は基本的に Pro プランを前提に進めるが、課金は必須ではなく無料体験や視聴だけでも受講できる
  • 料金はドル建て表示のことがあり、日本円換算は為替レートで変わるため目安として捉える
  • 料金・機能・利用条件は日々変わるので、契約前に必ず公式ページの最新情報を確認する
  • 会社・組織で使う場合は料金だけでなくプライバシーポリシーやデータの扱いを必ず確認する

Glossary

用語メモ

Pro / Max プランClaude の個人向け有料プラン。Pro は日々の作業向けで Claude Code を含み、Max はより多くの使用量を求める人向けの上位プラン。
Team / Enterprise会社・組織向けのプラン。Team は利用人数（シート）に応じて料金が変わり、Enterprise はより大きな組織向け。
シート法人向けプランで1人が使うための枠を指す。たとえば5枠を契約すれば5人が使えるイメージで、人数に応じて料金が決まる。
サブスクリプション月額や年額で継続して支払う利用契約のこと。Claude Code の利用には Pro 以上のサブスクリプションが必要。
プライバシーポリシー個人情報やデータをどう扱うかという方針をまとめた文書。入力したデータが学習に使われるかなど、特に会社利用時は事前確認が重要。
