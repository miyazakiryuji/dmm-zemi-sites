Episodes

各回の自習ガイド

EP01
コースの全体像とゴール

むずかしいコードは書かず、ふだんの日本語で頼みながらアプリやサイトを少しずつ仕上げていく——この入門コースが向かう先と全体像を、まずは最初に整理します。

EP02
Claude Codeとは

言葉で頼むだけでプログラムを組み立ててくれる「AIエンジニア」の正体を、Claudeの3モードと他社ツールとの違いから読み解きます。

EP03
Claude Codeでできること

Claude Codeの使いどころは「ウェブ制作」「アプリ開発」「業務効率化」の3つ。日本語で頼むだけでLPが組み上がる様子をデモで体感します。

EP04
料金プランの選び方

無料・個人向け・法人向けの違いを整理して、自分に合ったプランの判断材料をそろえます。

EP05
アカウント作成とプラン契約

Windows を例に、デスクトップアプリの導入からアカウント作成、プラン選択、最初の安全設定までを一気に通します。

EP06
Claude Codeの体験

「Hello world」を題材に、日本語で頼んで結果を受け取り、もう一度直してもらう——Claude Codeの基本の流れを手を動かして体感します。

EP07
ビジネス利用における注意点

Claude Code を仕事で安心して使うコツは、大きく2つ。秘密の情報を外へ出さないこと、そして判断を AI に丸投げしないことです。日々の作業で見返せる確認の視点までまとめました。

EP08
Claude Codeの基本操作

デスクトップアプリの画面を一周しながら、よく使うボタンと「指示→生成→確認」の流れを体で覚える回です。

EP09
バイブコーディングの基本

AIに日本語で話しかけながら開発を進める「バイブコーディング」の考え方と、つまずかないためのコツをやさしく整理します。
