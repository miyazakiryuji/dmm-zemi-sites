EP02
Claude Codeとは

言葉で頼むだけでプログラムを組み立ててくれる「AIエンジニア」の正体を、Claudeの3モードと他社ツールとの違いから読み解きます。

この回では、本コースを通じて使い続ける Claude Code が結局どういうものなのかを、まったく初めての方にも届くようにかみ砕いていきます。AI でコードを書く、と聞くと身構えてしまいがちですが、根っこにある発想さえ腑に落ちれば心配いりません。Claude Code がどこに位置づくのか、Claude に用意された3つのモード、そして似た役割を持つ他社ツールとの差を、ひとつずつたどっていきましょう。

Learn

この回の学習ノート

1Claude Codeは「AIエンジニア」のような存在

Claude Code を一言で表すなら、普段づかいの日本語で頼むだけでプログラムを組み立ててくれる、まるで AI のエンジニアです。専門的なプログラミングの素養がなくても、やりたいことを言葉にして伝えるだけで、アプリやツールの形に落とし込めるのが何よりの持ち味です。

進め方はいたって素直で、ざっくり3段階に分かれます。込み入ったコマンドを暗記する必要はなく、人に相談するのと変わらない感覚で開発を前に進められます。

  • 1. あなた自身が「こんなアプリが欲しい」「ここをこう変えたい」と日本語で依頼する
  • 2. Claude がその依頼を受け取り、必要なプログラム（コード）を書き起こす
  • 3. アプリやツール、Webサイトとして仕上がる
ネットで検索すると一見ハードルが高そうに映りますが、初学者でも順を追って取り組めば十分に手なずけられます。気負わず、まずは「AI にお願いして作ってもらう」という発想に体を慣らしていきましょう。

2Claudeの3つのモード（Chat / Cowork / Code）

意外かもしれませんが、Claude Code はそれ単体で完結する独立アプリではありません。ChatGPT や Gemini と並ぶよく知られた生成 AI アプリが Claude で、その中に目的別の3つのモードが用意されており、その一角を担うのが Claude Code です。

  • Claude Chat … おしゃべり・質問・相談ごとが得意なモード。発想の整理や下調べを助けてくれる
  • Claude Cowork … 手作業の肩代わりが得意なモード。資料づくりやファイルの整頓などを、まるで人が代行するように片付けてくれる
  • Claude Code … コーディングに振り切ったモード。Webサイトやアプリの制作に向く（本コースの主役）

じつは Claude Code と Cowork は、やっていることがほぼ地続きの間柄です。Claude Code はもともと、ターミナルと呼ばれる黒い操作画面（文字でコマンドを打って動かす、プログラマー向けの画面）で立ち上げて使うものでした。その同じ仕組みを、誰もが見慣れた画面のクリック操作からでも扱えるよう敷居を下げたのが Cowork、と捉えると分かりやすいでしょう。

本コースでは Claude Desktop アプリを入口に Claude Code を動かしていきます。黒い画面（ターミナル）と格闘する必要はなく、画面上の選択肢から呼び出せるので構えなくて大丈夫です。

3他のAIコーディングツールとの比較

AI とやり取りしながらコードを書かせていく開発のやり方は、Vibe Coding（バイブコーディング）と呼ばれます。この路線のツールは Claude Code だけではなく、よく名前が挙がるものに Codex と Antigravity があります。提供している会社、入口の作り、振る舞いの個性がそれぞれ違います。

  • Claude Code（Anthropic／Claude）… 入口はターミナルが基本だが IDE やクラウドからも使える。Claude の延長線上で気軽に呼び出せるのが強み。依頼の狙いを読んで自分から動く反面、頼んでいないことまで手を伸ばすこともある
  • Codex（OpenAI／ChatGPT）… ターミナル中心で IDE などからも扱える。指示には律儀だが慎重派で、細かく説明しないと意図を取りこぼすことも。エンジニアに支持されている印象
  • Antigravity（Google／Gemini）… 専用のエディタ（IDE）の上で動かすタイプ。無料の枠があるので試しやすく、Gemini だけでなく Claude も呼び出せる
ここでの良し悪しはあくまで個人的な所感で、各社のアップデートでころころ入れ替わります。一方が新機能を出せばすぐ他方も追いかける、そんな競り合いの真っ最中なので、この比較は「いま時点の目安」くらいに受け取ってください。

4なぜ本コースはClaude Codeを選ぶのか

数あるツールの中からあえて Claude Code を教材の中心に据えるのは、初学者にとって学びのとっかかりが多いから、というはっきりした理由があります。

  • 出来上がるコードの質と動きの安定感が高いと感じられること
  • 入門者向けの解説や日本語の作例にたどり着きやすく、つまずいても抜け出しやすいこと
  • Cowork と同様、Claude の画面から選ぶだけで気軽に立ち上げられること

そして見落としてはいけないのは、この回で身につくのは Claude Code という一つの道具の操作法だけでなく、その一段上にある Vibe Coding（AI コーディング）という考え方そのものだという点です。だからこそ、ここで育てた発想は Codex や Antigravity といった別のツールにもそのまま効いてきます。

ツール自体はこの先も変わり続けます。個々の手順を丸ごと覚え込むより、「AI に何を、どんな伝え方で頼めば狙い通りに動くか」という勘どころを養うほうが、どのツールでも持ち越せる地力になります。

Key Points

この回のポイント

  • Claude Code は、日本語で頼むだけでコードを書き起こしてくれる「AIエンジニア」のような道具で、プログラミングの知識がなくてもアプリやサイトを形にできる
  • Claude Code は独立したアプリではなく、Claude アプリに備わる3モード（Chat=対話・相談／Cowork=手作業の代行／Code=コーディング）のひとつ
  • Claude Code はもとはターミナル（黒い画面）で動かす道具で、それを画面のクリック操作で扱えるよう易しくしたのが Cowork というイメージ
  • 本コースでは Claude Desktop アプリを入口に Claude Code を使う
  • 他の Vibe Coding ツールには Codex（OpenAI）・Antigravity（Google）があり、提供元・入口・個性が異なる（評価は時期によって動く目安）
  • 本コースが Claude Code を選ぶ理由は、質と安定感、日本語情報の多さ、Claude から手軽に呼べる点。学びの核は Vibe Coding の考え方で、他ツールにも転用できる

Glossary

用語メモ

Claude CodeAnthropic が提供する、日本語などの指示からコードを書き起こすAIツール。Claude アプリに備わるコーディング向けのモードで、Webサイトやアプリの制作に向く。
Claudeの3つのモードClaude Chat（対話・質問・相談）、Claude Cowork（手作業の代行）、Claude Code（コーディング）の3つ。Claude Code はそのひとつで、本コースの主役となるモード。
Vibe Coding（バイブコーディング）AI とやり取りしながらコードを書かせていく開発スタイルの呼び名。Claude Code・Codex・Antigravity などがこの仲間にあたる。
ターミナル文字でコマンドを打ち込んで操作する黒い画面のこと。プログラマーがよく使う。Claude Code はもともとこの画面から起動・操作する道具だった。
IDE（統合開発環境）プログラマー向けのエディター。VS Code などが代表で、コードを書く作業に特化した環境を指す。
Codex / AntigravityClaude Code 以外の AI コーディングツール。Codex は OpenAI（ChatGPT）、Antigravity は Google（Gemini）が提供し、Antigravity では Claude も使える。
