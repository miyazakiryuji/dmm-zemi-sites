EP03
Claude Codeでできること

Claude Codeの使いどころは「ウェブ制作」「アプリ開発」「業務効率化」の3つ。日本語で頼むだけでLPが組み上がる様子をデモで体感します。

EP03では、Claude Codeが具体的に何に使えるのかを整理します。「コードを書くAI」と聞いても、自分の仕事にどう役立つかはイメージしづらいものです。そこで活用の幅を大きく3つに分けて押さえたうえで、実際にランディングページを作る短いデモで「日本語で頼む→できたものを見る→直したいところを伝える」という開発の流れをつかみます。

Learn

この回の学習ノート

1活用は大きく3つに整理できる

Claude Codeの用途は幅広いのですが、この回ではあえて「ウェブ制作」「アプリ開発」「業務効率化」の3本柱に整理します。最初にこの全体像を持っておくと、後の講義で具体的な機能を学ぶときも「これはどの柱の話か」と位置づけながら理解できます。

  • ウェブ制作 … 会社やお店の「見てもらう」ためのページを作る
  • アプリ開発 … ユーザーが「操作して使う」仕組みを作る
  • 業務効率化 … 毎日の面倒な作業を自動化する
「見てもらう」のがウェブサイト、「操作して使う」のがアプリ、と対比で覚えると、自分のやりたいことがどの柱に当てはまるか整理しやすくなります。

2ウェブ制作 ― 情報を見てもらうページを作る

1つ目はウェブ制作です。会社やお店の情報を伝えるためのページ作りに使えます。代表的な成果物は次の3つです。

  • ホームページ … 会社の情報をまとめて見てもらうためのページ
  • ランディングページ（LP） … サービスや商品の良さを1ページで紹介する集客向けのページ
  • モックアップ … 本番の制作に入る前に、見た目やレイアウトを試すための下書き的なデザイン

たとえば「自社サイトを一枚作ってほしい」と頼めば、Claude Codeは表示に欠かせないファイル一式を揃え、仕上がりのプレビューまで添えてくれます。デザインの作法を一から学ばなくても、まず言葉で頼むところからスタートできるのが魅力です。

3アプリ開発 ― 操作して使う仕組みを作る

2つ目はアプリ開発です。ブラウザで動くもの、スマートフォン向け、パソコンのデスクトップ向けと、いろいろな形のアプリ作りに対応できます。サイトが「眺めて情報を得る」ものだとすれば、アプリは「触って動かす」もの。入力された内容によって画面が切り替わったり、入れたデータがしまわれたり、裏で計算が走ったりと、何かしらの処理を経て答えが戻ってくる――そんなイメージです。

  • 予約システム
  • タスク管理ツール
  • 診断ツール
  • チャットツール
Claude Codeでのアプリ開発は、コードを一度書いてもらって完成、というものではありません。機能を足したい、この部分を直したい――そうした要望を伝えてやり取りを重ねるうちに、だんだん使えるアプリに育っていきます。

4業務効率化 ― 毎日の面倒な作業を自動化する

3つ目は業務効率化です。わざわざアプリを仕立てるほどでもないけれど、コツコツと続けていてうんざりするような作業――その手伝い役としてClaude Codeが活躍します。

  • ファイルやデータの分類・整理
  • 複数ファイルの一括リネーム（名前の変更）
  • 数値の集計やデータの転記
  • 情報をまとめて整理する自分専用ツールの作成

ExcelやGoogleスプレッドシートでの作業も、自動で片づけられます。これらの表計算ツールには以前から自動化の機能が備わっていて、Excel側は「マクロ」、Googleスプレッドシート側は「GAS」という名前で呼ばれます。決まった形の集計、レイアウトの整え、別の場所への書き写し、報告書づくりといった用途で重宝するものです。Claude Codeなら、この種の自動化スクリプトを書き起こしたり、手元の表とつなげたりしながら処理を進められます。

「毎日の同じ作業が面倒だな」と感じている人ほど、業務効率化の柱は効果を実感しやすい入り口です。まずは身近な手作業を1つ思い浮かべてみましょう。

5デモ ― 日本語で頼んでLPを作り、色を変える

ここからはClaude Codeが実際に手を動かす様子を見ていきます。題材は、会社を紹介するためのごく簡単なランディングページ。専門的な前提知識はいりませんので、ざっくり流れをつかむつもりで眺めてください。デモではメインで使っていくClaude Desktopアプリに対し、「シンプルな会社のランディングページを作ってください」と日本語でお願いします。

送信すると、Claude Codeがさっそくファイルを組み立て始めます。その途中で、書き込みの可否を尋ねる確認ポップアップが立ち上がります。これは画面を映し出すためのHTMLファイルを保存してよいかという確認なので、中身を見たうえで許可しましょう。一回ごとに聞かれると煩わしいため、今回のやり取り（セッション）はまとめて承認しておく、という選び方もできます。

承認するとHTMLファイルが書き出され、アプリに組み込まれたブラウザへページがそのまま映し出されます。社名やサービスの説明、強み、問い合わせ用のフォームに至るまで、ほんの一瞬で形になる手応えが味わえます。さらに「目を引くように青を基調にして」と続けて頼めば、配色がぐっと青へ切り替わります。

内蔵ブラウザに自動で映らないときは、「内蔵のブラウザを使って」とひとこと添えれば、ちゃんと表示してくれます。

このデモで覚えておきたいのは、欲しいものを日本語で頼み、仕上がりを見て、手を入れたい点を返す――という一連のサイクルです。このやり取りを何度も回しながら進めていくのが、Claude Codeの土台となる使い方になります。そして同じ進め方は、サイト制作に限らず、アプリづくりや作業を楽にするツール開発にもそのまま生きてきます。

Key Points

この回のポイント

  • Claude Codeの活用は「ウェブ制作」「アプリ開発」「業務効率化」の3本柱で整理できる
  • ウェブ制作はホームページ・ランディングページ（LP）・モックアップなど「見てもらう」ページ作り
  • アプリ開発はウェブ・スマホ・デスクトップなど多様で、対話を重ねて実用的にしていける
  • 業務効率化はファイル整理・一括リネーム・集計や、Excelマクロ・GASによる表作業の自動化に使える
  • デモではClaude Desktopアプリに日本語で指示し、会社紹介LPのHTMLを生成して内蔵ブラウザに表示
  • 「日本語で伝える→確認する→改善を伝える」の対話で開発を進めるのが基本の流れ

Glossary

用語メモ

ランディングページ（LP）サービスや商品の良さを1ページにまとめて見せる紹介用ページ。本体のホームページとは分けて、集客や案内を狙って用意されることが多い。
モックアップ本格的な制作に入る前に試しに作る下書きのデザインや画面のサンプル。仕上がりの見た目やレイアウトを事前に確かめる目的で使う。
ExcelマクロExcelで決まった手順の作業を機械任せにできる機能。集計や表の整形のような繰り返し作業を、プログラムに代わりにこなしてもらえる。
GAS（Google Apps Script）Googleスプレッドシートまわりの操作をプログラムで自動化できる仕組み。立ち位置としてはExcelのマクロのGoogle版にあたる。
HTMLウェブページの画面を表示するためのマークアップ言語。デモではこのHTMLファイルが生成され、ブラウザに表示された。
内蔵ブラウザClaude Desktopアプリに備わっている、作った画面をその場で表示・確認できるブラウザ機能。
