EP05
アカウント作成とプラン契約

Windows を例に、デスクトップアプリの導入からアカウント作成、プラン選択、最初の安全設定までを一気に通します。

Claude Code をいよいよ自分のパソコンで動かすための、最初のセットアップ回です。Claude のデスクトップアプリを入れ、その延長でアカウントを用意し、プランを決めて、初回チャット前の安全設定まで終わらせます。題材は Windows ですが、引っかかりやすい箇所と考え方そのものは、環境が違っても同じように効きます。

Learn

この回の学習ノート

1セットアップ前に知っておきたい前提

この回でやることは大きく4つです。(1) デスクトップアプリのインストール、(2) アカウント作成、(3) プランの選択、(4) 初回チャット前の安全設定。アカウントは単独で登録するわけではなく、アプリの初期設定を進める中で自然に出来上がる流れになっています。

Claude のアプリやサイトの見た目は、アップデートのたびにどんどん変わります。ボタンの場所、ページの作り、料金の出方などが、この解説と食い違って見えることはよくあります。

解説と画面がずれていても、慌てないことが肝心です。「自分で調べて先へ進む」練習だと割り切って、次のような手を一つずつ試してみてください。

  • 名前や意味の近そうなボタンを探してみる
  • そのページ全体にざっと目を通し、目的の項目が隠れていないか見る
  • ChatGPT や Claude（無料版でも構いません）に今の状況を伝えて尋ねる
  • Google 検索で最新のやり方を調べる
アプリの入れ方は、お使いの OS（Windows / Mac などパソコンを動かす土台のソフト）ごとに違ってきます。この回で扱うのは Windows 版です。Mac の方は、ここで示す流れを下敷きにしつつ、ご自身の環境に置き換えて読み進めてください。

2デスクトップアプリをインストールする（CPU の種類に注意）

ブラウザ（Chrome でも Edge でも、普段使いのもので大丈夫です）を開いて「Claude デスクトップアプリ」と検索し、上位に並ぶ公式サイト（claude.com）のダウンロードページへ進みます。検索結果には公式でないものが混ざる場合もあるため、配布元が公式かどうかは必ず見極めてください。

Windows 向けのダウンロードボタンには、通常版と Arm64 版の2つが用意されています。どちらを押すかは、自分のパソコンに入っている CPU（処理の中心を担う部品）の種類によって決まります。CPU の種類はおおむね X64 か Arm64 のいずれかで、世の中の Windows PC の大半は X64、ごく一部の新しい機種だけが Arm64 です。

どちらの CPU かは、Windows の設定画面からたどれます。確認の流れは以下のとおりです。

  • 画面下のスタートを右クリックして、設定のメニューを表示する
  • 「システム」を選ぶ
  • 「バージョン情報」の「デバイスの仕様」にある「システムの種類」を見る
  • そこに出ている値（たとえば x64 ベース）が、自分の CPU の種類
「システムの種類」の見方は、Claude だけでなく他のアプリを導入するときにも繰り返し使う、土台になる操作です。Windows を使う方は一度押さえておくと何かと役立ちます。

自分の CPU が分かったら、それに対応するインストーラーを落とします（X64 なら通常版、Arm64 なら Arm64 版）。落としたインストーラーをダブルクリックするとセットアップ画面が開くので、表示される案内のとおりに進めていきます。

3アカウント作成と、必ず確認したい規約・初期設定

セットアップを進めていくと、サインインの画面が現れます。アカウントをまだ持っていなければ、ここでメールアドレスを打ち込んで進むと、そのままアカウントが用意されます。Google（Gmail）でのログインを選んだ場合は、Google 側の認証画面でアカウントを指定してから先へ進む形になります。

アカウント作成の画面に出てくる利用規約とプライバシーポリシーは、リンクを開いて必ず中身に目を通してください。年齢の条件（18歳以上など）やデータの扱い方など、見落とせない事項が記載されています。

プロモーションメール（不定期のお知らせ）を受け取るかどうかは自由です。登録してもしなくても支障はありません。規約への同意を済ませた時点でアカウントが作られます。

アカウントが用意できると、最初のチャットに入る前に「AI モデルへの協力」を確認する画面が表示されます。これは、自分が交わしたチャットの内容を Anthropic 社の AI モデルの学習に役立ててよいか、を決める設定です。オンのままにしておくと、自分の会話が学習データとして使われることがあります。

プライバシーが気になるなら、この AI 学習への協力はオフにしておくと安心です。無料プランでもオフへ切り替えられるので、最初の段階でオフにしておくのがおすすめです。

続いてユーザー名を決め、何に使うか（仕事で使うかなど）を尋ねられます。利用目的は「後で設定する」で先送りにしても差し支えありません。これでアカウントの登録と最初の設定はひと通り終わりです。

4プランの選び方と、利用枠・従量課金の考え方

セットアップの途中でプランを選ぶ画面が出てきます。お金をかけずに始めたいなら Free を、はじめから有料で使いたいなら Pro などの有料プランを選びます。有料を選んだ場合は、月額か年額かを決め、クレジットカードを登録して契約を仕上げる流れになります。

プランは後から変更も可能です。デスクトップアプリの左下にあるアカウント表示や、上部に出る「アップグレード」の案内をたどれば、無料から有料へ乗り換えられます。

この講座の中心である Claude Code（と Cowork）は、無料プランでは動きません。実際に自分の手で試すには、有料プランの契約が前提になります。

有料プランだからといって青天井で使えるわけではない、という点には気をつけてください。各プランには「どのくらい使えるか」を表す利用枠が設けられていて、Claude Code を動かした分もここに数えられます。短い時間に集中して使いすぎると一時的に止まることがあり、Claude Code には週ごとの上限も別途あります（仕様は今後変わる可能性があります）。

利用枠を使い切ってもなお使いたいときの受け皿が「従量課金」です。これは月額プランに付いてくる枠とは切り離して、実際に使った分だけ料金が乗っていく方式です。押さえておきたいのは、上限に届いたからといって勝手に従量課金へ移るわけではない、という点です。あくまで自分が選んだぶんだけ上乗せで使える仕組みなので、気づかないうちに請求が膨らむ心配はありません。

有料プランは無理に入るものではありません。まずは講座をひと通り眺めてみて、「やはり自分の手で動かしてみたい」と思えたら、そのときに自分なりに納得して契約を考えれば十分です。

Key Points

この回のポイント

  • アカウントはデスクトップアプリのセットアップ中に出来上がる。入れ方は OS ごとに異なり、この回の題材は Windows 版
  • 画面はアップデートでしばしば姿を変える。違っていても、近い名前のボタンを探す・ページ全体を見る・AIや検索に尋ねる、で自力解決を試す
  • ダウンロード前に Windows の「システムの種類」で CPU が X64 か Arm64 かを見極め、それに合うインストーラーを選ぶ（たいていは X64）
  • アカウントを作るときは、利用規約とプライバシーポリシーを必ず開いて中身を確かめる
  • 最初のチャットの前に、Anthropic の AI モデル学習へのチャット提供は（無料プランでも）オフにしておくと安心
  • Claude Code と Cowork は無料プランでは動かず有料プランが要る。利用枠には上限があり、超えた分は自動ではなく任意の従量課金で使える

Glossary

用語メモ

OSOperating System の略。Windows や Mac など、パソコンを動かす土台のソフトを指す。アプリの入れ方は OS ごとに変わる。
CPU（X64 / Arm64）CPU は処理の中心を担う部品で、X64 と Arm64 はその種類。Windows PC の多くは X64、新しい一部機種が Arm64。種類に合うインストーラーを選ぶ。
AI 学習への協力（オフ設定）自分のチャット内容を Anthropic 社の AI モデルの学習に役立てるかを決める設定。最初のチャットの前に出てきて、無料プランでもオフにできる。
プラン（Free / Pro / Max）Claude の料金プラン。Free は無料、Pro・Max は有料。Claude Code と Cowork は無料プランでは動かず、有料プランの契約が要る。
従量課金月額プランの利用枠とは別枠で、実際に使った分だけ料金が乗っていく方式。上限を超えて使うための受け皿で、勝手に課金されることはない。
