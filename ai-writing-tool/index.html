Claude Code で
生成AIライティングツールを作る

「作りたいもの」を言葉で伝えるだけで、Claude Code がアプリを生成します。EP19 の手順を、指示 → 生成 → 起動 → 確認の流れで整理し、補足を加えました。EP19＝教材の手順／補足＝発展メモ。

Goal

今回作るものEP19

複数のライティング補助AIを1つにまとめた、自分専用の「AIライティングツール」。最終的に公開はしますが、他のユーザーが使う想定はありません（個人用）。

✍️ ブログ記事執筆を手伝う
📧 メール返信返信文を考える
📝 文章要約長文を短く
🔁 リライト言い換え・調整
🌐 翻訳多言語へ

💡 補足

  • 「個人用・他ユーザー利用なし」なので、ユーザー認証やデータベースは作りません（シンプルに保てます）。
  • 前提知識は Python / Streamlit / API 入門 を見ておくと、何を作っているか分かりやすくなります。

Tech Stack

使う技術EP19

  • Python＝処理を書く言語
  • Streamlit＝Pythonだけで画面（UI）まで作れる道具
  • Gemini API＝文章を生成してくれるAI（無料枠あり）
  • 今回は個人用なので データベース・認証は不要

💡 補足

  • 3つの役割と関係（3層構造）は こちらの図 が分かりやすいです。
  • Gemini を選ぶ理由は無料枠があるから。APIキーの発行は次以降（EP21）で扱います。

Hands-on

作る手順EP19

Claude Code に「作りたいもの」を言葉で伝えるところから始めます。

1

Claude Code に「作りたいもの」を指示する

音声入力（Aqua Voice）でも、キーボード入力でもOK。次のように、作りたいもの・使う技術・要らないものを伝えます。

これから AIライティングツール を作っていきます。個人用で、たとえばブログ記事を執筆してくれるAI、メールの返信文を考えてくれるAI、文章を要約してくれるAI、こういったものを1つにまとめたツールにしたいです。便利なライティング系のAI機能があれば適当に追加しても構いません。
利用する技術スタックは Python・Streamlit・Gemini API を使います。
今回はデータベース・認証なしの個人用で作るので、それらは不要です。

💡 補足：Plan モード / CLAUDE.md は今回は後回しでOK

  • Plan モード（綿密に計画してから実装）は今回のような単純なアプリでは不要。使いたい時は Shift + Tab でモードを切り替えられます。
  • CLAUDE.md（ルール書き）は後で作るのがオススメ。最初にルールを決めなくても、ある程度の雛形ができてから作る方が、特に非エンジニアの方には効率的です（CLAUDE.md の作成は EP20）。
2

ファイルが生成される

Claude Code が app.py（アプリ本体）や requirements.txt（必要なライブラリの一覧）などを1分以内で生成します。

💡 補足

  • ファイルの意味が分からなくても大丈夫。「このファイルはどういう意味ですか？」と Claude Code に聞きながら学ぶのが一番の近道です。
  • requirements.txt＝このアプリを動かすのに必要な“部品（ライブラリ）の買い物リスト”のようなものです。
3

Streamlit でアプリを起動する

新しいターミナル（ターミナル上部のプラス「＋」ボタン）を開いて、次の2行を実行します。

pip install -r requirements.txt
streamlit run app.py

Local URL: http://localhost:8501 が表示されたら、そのURLをブラウザに貼り付けるか、Ctrl＋クリック（Mac は Cmd＋クリック）で開きます。

💡 補足

  • pip install -r requirements.txt＝必要なライブラリをまとめて一括インストールするコマンド。
  • localhost:8501 は自分のPCの中だけで動くURLです。まだインターネットには公開されていません（世界に公開するのは後の「公開」回）。
4

動作を確認する

ブラウザに「AIライティングツール」のページが開きます（ブログ記事・メール返信・要約・リライト・翻訳など）。

⚠️ 補足：まだ生成は動きません

  • この時点では Gemini API キーが未設定なので、文章生成はまだ動作しません。画面（UI）が表示されればOKです。
  • APIキーの発行・設定は次のステップ（EP21）で行います。

Troubleshooting

困ったら：サイドバーが出ない場合EP19

環境によっては、左サイドバー（APIキー入力欄など）が出ない実装になっている場合があります。そのときは Claude Code にこう頼めば追加されます。

サイドバーを作ってください。
Gemini API キーの入力フォームも配置してください。

💡 補足

  • うまくいかない時の基本は 「画面をスクショして、何が起きたかを添えて Claude Code／生成AIに聞く」。詰まったら気軽に。
