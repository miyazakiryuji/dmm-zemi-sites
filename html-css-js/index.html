EP13 ・ Webの3点セット

同じページを
HTML → ＋CSS → ＋JS と
1枚ずつ重ねて見る

Webページは HTML（骨格）・CSS（見た目）・JavaScript（動き）の3つでできています。 同じ1枚のページを3段階で実際に表示して、どの層が何をしているのかを見比べてみましょう。

The Three Layers

Webの3点セット：役割だけ覚えればOK

文法を覚える必要はありません（コードは Claude Code が書いてくれます）。大事なのは「どれが何の担当か」だけ。この3つの“役割”を押さえておきましょう。

HTML

骨格

見出し・文章・画像・ボタンなど、ページの“中身（情報）と構造”を作ります。家でいう柱や間取り。

CSS

見た目

色・余白・文字の大きさ・配置など、“デザイン（装飾）”を担当。同じ骨格でも印象が一変します。内装や塗装。

JavaScript

動き

クリックで反応する・数が増える・開け閉めするなど、“動き（インタラクション）”を追加。電気やスイッチ。

Live Switch

ボタンで切り替えて、同じページを見比べる

下のボタンで、同じ1枚のページに CSS と JS を一枚ずつ重ねていきます。中身（HTML）は3つとも全く同じ。何が変わるかに注目してください。

penguin-shop.example ／ ① HTML

Summary

    
まとめ：役割で覚える3点セット

    

      
      
    
        
  • 
          
          HTML ＝ 骨格。見出し・文章・画像・ボタンなど、ページの中身と構造をつくる。
        
    • 
        
  • 
          
          CSS ＝ 見た目。同じHTMLでも、色・余白・配置を付けて読みやすく美しくする。
        
    • 
        
  • 
          
          JavaScript ＝ 動き。クリックで反応・数が増える・切り替わるなど、ページを“反応”させる。
        
    • 
        
  • 
          
          文法は覚えなくてOK。「どれが何の担当か」さえ分かれば、直したい時に「CSSで色を変えて」のようにClaude Code に頼める。※ このページ自体も HTML・CSS・JavaScript の3点セットでできています。
