Key Points

今回押さえる3つのポイント

LIVE講義でも重点的に解説する観点です。自習時の視聴メモの目安にもどうぞ。

1

Node.js → Claude Code の順は必須セット。バージョン確認まででワンセット

Claude Code は npm 経由で入れるので、Node.js を先に入れていないと npm install 自体ができません。入れたら必ず node -v / npm -v でバージョンが表示されるところまで確認。Node.js インストール後はエディタを一度落として開き直す必要があるのも詰まりやすいポイントです。

2

初回ログインは Subscription 認証 (Pro $20〜) がおすすめ

ログインは3択（Subscription / Anthropic Console API / その他）。API従量課金は使うほど課金されるので、初心者は Pro $20プラン から始めるのが安全。「5時間ごとリセット」「週次制限」はWeb版 Claude の「使用量」画面で確認できます。本格運用に切替えるなら Max $100 以上。

3

Claude Code はフォルダ単位で動く。権限は .claude/settings.local.json に蓄積

Web Search / Fetch / Edit などの権限ダイアログで 「2. 常に許可」 を選ぶと、そのフォルダの .claude/settings.local.json に permissions.allow として記録されます。フォルダごとに別の権限になるので、用途別にプロジェクトフォルダを切る発想が大事。EP09 では claude-code-tutorial/lesson1 のように切る例を扱います。

When You're Stuck

分からないことが出てきたらこの順番で

学習中・課題実施中に詰まったら、以下のフローで進めてください。1人で抱え込まないことが一番大事です。今回は環境構築回なので、特に詰まりやすいです。気軽に投げてください。

1STEP 1

まずは
スクショしてAIに聞く

エラー画面・つまずいた画面を スクリーンショット で撮り、Claude／ChatGPT／Gemini など普段使っている生成AIに貼り付けて質問してみてください。

精度を上げるコツ：
  • 何をしようとしていたか
  • どうなる予定だったか
  • 実際にどうなったか
をスクショに添えて投げる。

2STEP 2

それでも解決しなければ
ゼミ掲示板へ

AIに聞いても解決しない／そもそも何を聞けばいいか分からない場合は、DMM 生成AI CAMP 学び放題 の上部ナビゲーション 「ゼミ掲示板」 から投稿してください。他の受講生・運営・講師が回答できます。

投稿テンプレ：
  • 実現したかったこと
  • 試したこと（STEP 1 のAIとのやり取りも含めるとなお良い）
  • つまずきポイント（スクショ歓迎）
  • 環境（OS／ツール／バージョン）

「これは聞いていいレベルか…」と迷う必要はありません。STEP 1 → STEP 2 の順番で気軽に出してください。
